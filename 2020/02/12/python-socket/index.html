send与sendall的区别和使用方法

  • send(string[, flags]) 发送TCP数据，返回发送的字节大小。这个字节长度可能少于实际要发送的数据的长度。换句话说，这个函数执行一次，并不一定能发送完给定的数据，可能需要重复多次才能完成发送。
  • sendall(string[, flags]) 发送完整的TCP数据，成功返回None，失败抛出异常。
